3막 구조 · 스토리라인 개요

ACT 1 · 문제 설정 (제안개요 + 점검내용)
"축평원의 정보화 사업, 무엇이 위험한가?"
딥서치 근거: 2024년 농식품부 종합감사 보안 규정 위반 적발, 3개 사업(210~240일) 통합 일정 리스크, 스마트축산본부 신설(2026.01)에 따른 정보화 투자 확대 기조, 신임 원장(박수진)의 '데이터 기반 축산 행정 지능화' 경영방침.
→ 문제 프레이밍: 단순 통합감리가 아니다. 2024년 보안 지적 재발 방지 + 3개 사업 일정 리스크 통제 + 신임 원장의 디지털 혁신 의지를 뒷받침할 컨설팅형 통합감리가 필요하다.
ACT 2 · 해법 제시 (감리방법 + 점검방법 + 감리일정)
"1인 법인의 차별화된 통합감리, 어떻게 실현할 것인가?"
핵심 메시지: "단일 의사결정 체계의 기동성 + 15년 경력 수석감리원 현장 직접 수행 + 5개 영역 외부 전문가 Pool = 품질 높이고 비용은 낮춘 통합감리"
3대 솔루션:
① 보안 3중 점검 체계 — 보안약점진단(국가정보원 인증 도구) + 개인정보보호 진단(S등급 유지 지원) + 물리적 보안 점검 → 2024 감사 지적 재발 방지
② 통합 WBS 기반 일정·품질 통제 — 3개 사업(기간 상이) 통합 마일스톤 관리, 단계별 게이트 체크
③ 데이터 품질·표준화 점검 — 빅데이터 플랫폼 고도화 대비 DB표준화지침 제10조 기반 2레벨 체크리스트
ACT 3 · 실행 역량 (감리인력 + 사업관리 + 기타지원)
"왜 우리가 이 통합감리를 수행해야 하는가?"
차별화 근거:
① 상근 100% — RFP 요건(50%↑)을 상회하는 1인 법인 상근 체계. 감리 품질 일관성 확보
② 축산 IT·빅데이터 전문성 — 외부 전문가 Pool(빅데이터·AI·축산정보) 보유. 스마트축산·빅데이터 특화 감리 가능
③ 자동화 도구 3종 보유 — 보안약점진단도구 + 웹접근성 자동평가도구 + 기능점수(FP) 측정도구
④ 컨설팅형 감리 — 단순 부적합 지적이 아닌 '사업 완수 지원' 감리 철학. 신임 원장(박수진)의 IT서비스 공로공무원상(2026.05) 수상 경험과 연계한 실질적 IT 컨설팅 제공

Pain Point → 솔루션 매핑

PAIN POINT 01
🔴 2024 농식품부 감사: 보안 규정 위반 재발 위험
근거: 브릿지경제(2024.05.16) — 보호구역 출입 사후결재/무결재, 복무기강 해이, 자의적 규정 운영 적발. 처분은 주의·개선에 그쳐 재발 가능성 상존.
영향: 감리대상사업 3개 모두 정보시스템 → 보안 취약점 내재 시 감사 재적발 위험
SOLUTION 01
🟢 보안 3중 점검 체계 제안
제안서 메시지: "2024년 감사 지적의 재발을 방지하는 3중 보안 점검 — ① SW 보안약점진단(국정원 인증 도구) ② 개인정보보호 진단 ③ 물리적·관리적 보안 점검"
차별화: 단순 지적이 아닌 '재발 방지 체계'를 제안서 전면에 배치. 진단도구 + 진단원 자격 + 점검 프로세스 원패키지
PAIN POINT 02
🔴 3개 사업(210~240일) 통합 일정 조정 리스크
근거: RFP — 사업1(계열화+e음, 240일), 사업2(수급플랫폼4단계, 210일), 사업3(빅데이터고도화3차, 210일). 기간 상이, 단계별 감리 시점 상이 → 통합 일정 복잡도 높음
영향: 감리 투입 시점 오차 → 단계별 감리 공백 또는 중복 발생
SOLUTION 02
🟢 통합 WBS 기반 마일스톤 관리
제안서 메시지: "3개 사업의 기간 차이를 극복하는 통합 WBS. 단일 의사결정 체계(홍부장 직접 PM)로 신속한 일정 조정. 단계별 게이트 체크로 일정 이탈 사전 차단"
차별화: 1인 법인의 '단일 의사결정' 강점. 대형 법인 대비 의사결정 속도 3배↑
PAIN POINT 03
🔴 스마트축산·빅데이터·AI 특화 감리 전문성 부족
근거: 스마트축산본부 신설(2026.01, 농민신문), AI 양돈 생산성(2024~2025), 빅데이터 표준화 가이드북(2024.12). 박수진 원장 IT서비스 공로공무원상(2026.05) — IT/데이터 전문성 강조
영향: 일반 감리법인은 축산 IT 특화 영역 감리 경험 부족 → 전문성 검증 필요
SOLUTION 03
🟢 축산 IT·빅데이터 전문 감리 Pool 운영
제안서 메시지: "축산·농림 분야 IT 감리 경험 보유 감리원 + 빅데이터/AI 전문 외부 Pool. 스마트축산 3개 사업(계열화+e음, 수급플랫폼, 빅데이터) 각각의 기술 특성을 반영한 맞춤형 감리계획"
차별화: 일반 감리 + 축산 IT 특화 = 차별화된 가치 제공
PAIN POINT 04
🔴 1.7억 예산 대비 적정 감리 품질 확보
근거: RFP — 감리예산 170백만원. 3개 대상사업 추정액 30~56억. 감리비율 3~5.7%로 통상(3~8%) 대비 중하회. 효율적 통합감리 운영 필요
영향: 예산 대비 과다 공수 제안 시 수익성 악화, 과소 공수 시 감리 품질 저하
SOLUTION 04
🟢 효율적 통합감리 모델 + 원격·현장 하이브리드
제안서 메시지: "1인 법인 최적화된 통합감리 모델: 상근 핵심 + 외부 전문가 Pool + 원격 20% 병행 = 품질 유지 비용 최적화. MD당 단가 경쟁력 확보로 예산 내 최대 효과"
차별화: 대형 법인 대비 관리비 30%↓. 동일 예산 대비 더 많은 감리 포인트 점검 가능
